Tutorial Introduction

In the following several pages, we will give you tutorials on the three versions of our chatbot.

The idea of chatbots is simple: you ask a question, the bot will return answers. The only difference between the three versions is the format of responses. Specifically, the three versions are:

  1. Text-Bot: the response is pure text.
  2. Vis-Bot: the response includes text and graphic visualization (e.g. charts, graphs).
  3. InteractiveVis-Bot: the response includes text and interactive visualization (e.g., you can click on or hover over graphic elements).

Please read the instruction on the left of each page before trying out the chatbot on the right!

Now, please click “next section” to start the tutorials for our chatbots.

Text-Bot Tutorial

Please read this introduction before using the chatbot interface.

The text-bot interface is shown on the right.

After clicking a recent game button you are interested in, you can ask the bot any stats-related question.

Click enter on your keyboard or the send button to send the message.

Our bot currently supports stats info including:

  1. Player game stats: including all typical stats types such as points, rebounds, fouls, shot accuracy, etc.
  2. Team's game stats: including all regular stats type such as turnover, points, assits, shots made/missed etc.
  3. Player and team historical stats: including season-high/low/average for all typical stats types.

Now, please briefly try out the interface and get to know how to ask a question and get/read the responses. You will have chance to experience more later.

Don't know what to ask? try "who scored the most points in each team?"

Visualization Tutorial

Visualization, as the graphic representation of data, aims to communicate information clearly and efficiently to users. To that end, here we also present basketball game stats data in visuals to communicate the stats information.

In this section, we will show you some visualizations used in the Visual-Bot and InteractiveVisual-Bot.

Please carefully go through all four visualizations on the right. You can click “next section” to proceed if you can fully understand these visual charts.

The flow chart indicates the changes of the two teams' points with time.
The shot chart includes all the shots in a spatial layout.
The bar chart shows all the Lakers' players' points in this game.
The bar chart shows Lakers' season record so far: including home / away and win / lose.
